# Habit Tracker Web App

A GitHub-style habit tracking web app built with pure HTML, CSS, and JavaScript.

## Features
- Year & Monthly calendar views
- Streak tracking
- Stats dashboard
- PWA support (installable)
- Dark/Light theme
- Data backup & restore
- Export stats as TXT
- Share stats as image
- Daily reminder notifications

## Tech Stack
Vanilla JS, CSS Grid, LocalStorage, PWA Manifest

## How Data Works
All data is stored locally in the browser using LocalStorage. No server or account required.

## Run Locally
Just open `index.html` in a browser.
